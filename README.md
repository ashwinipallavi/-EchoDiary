# EchoDiary 📖

EchoDiary is a simple and secure mobile diary application built using **Flutter** and **Firebase**.  
It allows users to write, store, and manage their personal diary entries safely in the cloud.

This project was developed as part of a **Hackathon project** to demonstrate mobile app development with real-time cloud storage.

---

## 🚀 Features

- ✍️ Create and save personal diary entries
- ☁️ Cloud storage using Firebase
- 🔐 Secure user authentication
- 📱 Clean and simple mobile UI
- 📝 Edit and manage diary notes easily

---

## 🛠 Tech Stack

- **Flutter**
- **Dart**
- **Firebase Authentication**
- **Firebase Firestore / Realtime Database**
